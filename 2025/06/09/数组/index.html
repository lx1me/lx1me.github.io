数组

概念:

指的是一种容器，可以同来存储同种数据类型的多个值。

但是数组容器在存储数据的时候，还需要结合隐式转换考虑。

比如：

定义了一个int类型的数组。那么boolean。double类型的数据是不能存到这个数组中的，

但是byte类型，short类型，int类型的数据是可以存到这个数组里面的。

建议：

容器的类，和存储的数据类型保持一致。

举例：

整数1 2 3 4 56 就可以使用int类型的数组来存储。

小数1.1 1.2 1.3 1.4 就可以使用double类型的数组来存储。

字符串”aaa” “bbb” “ccc” 就可以使用String类型的数组来存储。

数组的定义

格式一:数据类型 [] 数组名

例: int [] array

格式二:数据类型 数组名 []

例: int array []

方括号和数组名谁前谁后都一样.

数组的静态初始化

代码格式:

数据类型[] 数组名 = new 数据类型[]{元素1，元素2，元素3…};

例:int[] array = new int[]{1,2,3…};

int[] array = {1,2,3…};

格式解释:

数据类型：限定了数组以后能存什么类型的数据。

前面和后面的数据类型一定要保持一致。

方括号：表示现在定义的是一个数组。

数组名：是一个名字，方便以后使用。

大括号：表示数组里面的元素。元素也就是存入到数组中的数据。

多个元素之间，一定要用逗号隔开。

注意点:

①等号前后的数据类型必须保持一致。

②数组一旦创建之后，长度不能发生变化。

地址值

打印数组的时候，实际出现的是数组的地址值。

数组的地址值：就表示数组在内存中的位置。

比如:

[: 表示现在打印的是一个数组.

I: 表示答应的数据类型是int类型.

@: 是一个间隔符号.

1540e19d: 表示是数组在内存中真正的地址值.(十六进制)

∴ [I@1540e19d 就是数组的地址值

数组元素的访问

代码格式:数组名[索引]

作用:①获取数组中对应索引上的值

②修改数组中对应索引上的值

一旦修改之后，原来的值就会被覆盖了。

索引的特点:

  1. 一定是从0 开始的
  2. 连续不间断
  3. 逐个+1增长

数据的存储

代码格式:数组名[索引]=具体数据/变量

遍历：就是把数组里面所有的内容一个一个全部取出来。

代码格式:数组名.length;

2.如果是偶数，则将当前数字变成二分之一

数组的动态初始化

代码格式:数据类型[] 数组名 = new 数据类型[数组的长度];

数组的默认初始化值：

整数类型：0

小数类型：0.0

布尔类型：false

字符类型：’\u0000’

引用类型：null

数组两种初始化方式的区别

静态初始化：int[] arr = {1,2,3,4,5};

动态初始化：int[] arr = new int[3];

静态初始化：手动指定数组的元素，系统会根据元素的个数，计算出数组的长度。

动态初始化：手动指定数组长度，由系统给出默认初始化值。

使用场景:

只明确元素个数，但是不明确具体的数据，使用动态初始化。

已经明确了要操作的所有数据，使用静态初始化。

数组的内存图

1.JAVA内存分配:

栈:方法运行时使用的内存，比如main方法运行，进入方法栈中执行

堆:存储对象或者数组，new来创建的，都存储在堆内存

方法区:存储可以运行的class文件

本地方法栈:JVM在使用操作系统功能的时候使用，和我们开发无关

寄存器:给CPU使用，和我们开发无关

注:只要是new出来的一定是在堆里开辟了一个小空间

如果new了多次,那么在堆里有多个小空间,每个小空间有各自的数据.

注:

当两个数组指向同一个小空间时，其中一个数组对小空间中的值发生了改变，那么其他数组再次访问的时候都是修改之后的结果了。
